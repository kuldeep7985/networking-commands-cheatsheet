# 🌐 Networking Commands Cheatsheet
Quick reference for IT Support & Network Engineers

## Windows Commands
ipconfig /all         — Show all IP info

ping 8.8.8.8          — Test internet connection

tracert google.com    — Trace network path

nslookup google.com   — Test DNS resolution

netstat -an           — Show open connections

arp -a                — Show ARP table

## Linux Commands  
ip addr show          — Show IP addresses

ping -c 4 8.8.8.8     — Test connectivity

traceroute google.com — Trace route

nslookup google.com   — DNS lookup

netstat -tuln         — Open ports

ss -tuln              — Socket statistics

## Cisco Commands (Packet Tracer)
show ip int brief     — Interface summary

show vlan brief       — VLAN summary

show ip route         — Routing table

show running-config   — Current config
